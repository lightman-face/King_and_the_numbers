# 七王五二三

一个双人回合制纸牌游戏，包含网页联机版和 Tkinter 本地版，均无需安装第三方依赖。

## 网页联机版（推荐）

```powershell
python -B web_server.py
```

然后用浏览器打开 `http://localhost:8000`。一名玩家创建房间并分享六位房间码，另一名玩家在同一网址输入房间码加入。

同一 Wi-Fi 下，可让另一台设备打开 `http://房主电脑的局域网IP:8000`。Windows 可运行 `ipconfig` 查找 IPv4 地址；首次启动时需要允许 Python 通过 Windows 防火墙的专用网络。

若要通过互联网邀请朋友，需要将项目部署到公网服务器。仓库已包含 `render.yaml`，可直接用于 Render 部署。部署后会得到一个固定的网址，任何人都可以通过该网址打开游戏界面，再通过房间码加入同一个房间。

## 部署到公网
1. 访问 https://render.com 并登录或注册账号。
2. 选择“New Web Service”，连接你的 GitHub 仓库。
3. 选择本仓库，Render 会读取根目录下的 `render.yaml`。
4. 直接创建服务，Render 会使用 `python -B web_server.py` 启动。
5. 部署完成后，Render 会给出一个固定的公网地址，比如 `https://your-service.onrender.com`。

访问这个固定地址即可进入游戏界面。不同网络环境的玩家只要都访问这个地址，并输入同一个房间码，就可以进入同一局游戏。

> 注意：当前后端把房间数据保存在服务器内存里，Render 的免费实例或重启后会清空房间；正在进行的牌局会结束。

## 本地同屏版

```powershell
python main.py
```

## 操作方式

- 两位玩家共用一台设备并轮流操作。
- 轮到某位玩家时会显示隐私遮罩；接过设备并确认后，只会显示自己的手牌。
- 点击手牌进行选择，再点击“出牌”。玩家也可在对方出牌后选择“放弃本回合”。
- 每回合胜者收取桌面上的 5、10、K 作为计分牌，并在下一回合先补牌、先出牌。

## 当前规则约定

- 牌力顺序：7、大王、小王、5、2、3、A、K、Q、J、10、9、8、6、4。
- 同点数按黑桃、红桃、梅花、方片排序。
- 普通压牌必须同类型、同张数；炸弹（三张同点数）可压单牌或对子。
- 对子与炸弹同点数时，以其中最大花色比较。
- 打出由 7、王、5、2、3 组成的五张牌立即获胜，大小王均可。
- 牌堆抽完后，任一玩家出完手牌即按计分牌结算；K/10 每张 10 分，5 每张 5 分。

> 说明：这是本地“交接设备”双人版，不是网络联机版。
